Cartella per le foto dello slideshow della card "Focus Italia".

Carica qui le tue foto in qualsiasi formato (JPG, JPEG, PNG, WebP).
Puoi chiamarle come vuoi — non c'è nessun vincolo sui nomi.

Poi aggiorna il file manifest.json con i nomi esatti dei file che hai caricato:

Esempio manifest.json:
[
  "delusione-bosnia.jpg",
  "tifosi.png",
  "nazionale.JPG",
  "mio-nome-preferito.jpeg"
]

Formati supportati: .jpg .jpeg .png .webp (maiuscolo o minuscolo, entrambi ok).
L'ordine nel manifest determina l'ordine di visualizzazione.
Le foto vengono mostrate in loop, con dissolvenza ogni 12 secondi
e un leggero effetto Ken Burns (zoom lento + piccolo movimento).
